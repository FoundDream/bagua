# 千里命稿 — 核心理论

> 民国·韦千里著，以现代视角诠释古典子平，案例丰富，格局论命精辟，行运分析实用，是近现代命理学的重要桥梁著作。

---

## 一、命理总论

**韦氏核心观点**：
> "命理之学，不在神秘，在于五行生克之理，格局用神之法，因势利导，推断人生轨迹。"

**论命三要**：
1. **格局**：决定命局贵贱层次
2. **用神**：格局成败之关键
3. **行运**：命局发挥之时机

> 格局是骨架，用神是灵魂，行运是机遇。三者俱佳，方为上命。

---

## 二、格局精论

### 格局成败的四大要素

**一、用神有力**：
- 用神透出天干，有地支为根
- 用神得月令气势，或得三合、方合之助
- 用神不被合化、刑冲

**二、格局纯粹**：
- 单一格局，无混杂（如官格不见杀，财格不见劫）
- 若官杀混，须合去一方；若财劫混，须制去劫财

**三、喜神得力**：
- 格局所需之辅助（喜神）透出有力
- 喜神与用神相生相辅

**四、忌神受制**：
- 格局之忌神被合去、被冲掉、被克制
- 或忌神入墓、坐空亡

---

### 各格精要（韦氏总结）

**正官格**：
- 最喜财印相辅，官星清纯
- 见伤官则格破，见七杀则混局
- 名言：**"官星明现，名利双收；官星入库，仕途蹭蹬。"**

**七杀格**：
- 杀本凶，制化得宜则威权赫赫
- 制杀不宜过重（制多则无威）
- 名言：**"七杀逢制，英雄本色；七杀无制，横行霸道。"**

**财格**：
- 财旺须身旺，方能守财享财
- 财轻忌比劫夺，财重忌官杀得势资杀
- 名言：**"财多身弱，富屋贫人；财少身强，守财奴命。"**

**食神格**：
- 食神泄身，主口福才艺
- 食神制杀，威权自现
- 最忌枭印夺食
- 名言：**"食神有寿，身安福厚。"**

**伤官格**：
- 伤官佩印，才华横溢而有节制
- 伤官生财，才艺转化为财富
- 伤官见官，官非病灾（金水例外）

**印格**：
- 印旺用财（以财破印为用），反贵
- 印轻用官（官印相生），亦贵
- 印多身旺，须泄身（食伤为用）

## 三、用神取法（韦氏总结）

### 第一步：定日主强弱

| 判断项目 | 得分参考 |
|---|---|
| 月令是否同气（得令）| 最重要 |
| 地支是否有根（长生、帝旺、建禄）| 重要 |
| 天干是否有比劫、印星帮扶 | 次重要 |

### 第二步：看格局

- 月令透出之字为格局
- 月令藏干中最旺之气为格局依据

### 第三步：定用神

**身强**（用神原则：泄、耗、克）：
- 官杀克身 → 官杀为用
- 财星耗身 → 财星为用
- 食伤泄身 → 食伤为用

**身弱**（用神原则：生、助）：
- 印星生身 → 印星为用
- 比劫助身 → 比劫为用

**从格**：日主无根，顺从最旺之势（详见滴天髓）

**化格**：天干合化成功，以化神为用

### 第四步：调候辅助

- 参穷通宝典，根据月令寒热调配辅助用神

## 四、行运分析

### 大运推命口诀

> "大运如流年之根，流年如大运之枝。运好年差，小有波折；运差年好，难以挽回。"

**大运十年分看法**：
- 前五年看天干，后五年看地支
- 天干地支一致：力量纯粹，应验强烈
- 天干地支相克：前后有矛盾，生活起伏

**行运好坏判断**：
| 大运类型 | 结果 |
|---|---|
| 运扶喜神、用神 | 发福发贵 |
| 运制忌神 | 化险为夷 |
| 运克用神 | 破格，灾祸 |
| 运助忌神 | 凶事临门 |
| 运入空亡 | 运势虚浮，难有实效 |

### 流年应期

**应期判断**：
- 流年天干与命局天干形成六合 → 感情、合作之事
- 流年地支冲命局地支 → 动荡、变化、搬迁
- 流年与大运同气 → 应验最强
- 流年逢空亡 → 该年计划难以实现

**流年断事口诀**：
- 流年财星临 → 财运、婚配
- 流年官星临 → 升职、考试、官非
- 流年印星临 → 学业、文件、母亲
- 流年食伤临 → 才艺、子女、口舌
- 流年比劫临 → 竞争、兄弟、破财

## 五、六亲推断（实用总结）

### 婚姻

**男命看妻**：
- 正财为正妻，偏财为外遇或再婚
- 日支为妻宫（最直观）
- 财星逢合：妻有情缘，或妻主动
- 财星被冲：婚姻不稳，易有风波
- 财星入墓：妻体弱或缘分有限

**女命看夫**：
- 正官为夫星
- 官星清纯有力 → 丈夫有作为
- 官杀混杂 → 感情复杂，婚姻不稳
- 官星逢合（被他干合去）→ 丈夫有外遇或不专一
- 官星入墓 → 丈夫郁郁不得志

**晚婚迹象**：
- 日支逢空亡
- 财星（男）或官星（女）坐空亡
- 桃花无力或华盖重重
- 命局孤鸾煞并见

### 子女

- 男命以官杀为子女：官为女，杀为男
- 女命以食伤为子女：食神为女，伤官为男
- 时柱为子女宫，时支坐空亡或冲克 → 子女缘薄
- 枭印夺食（女命）→ 子女少或与子女缘分淡

### 财富

**富命特征**：
- 财星得地有力，日主有根能守财
- 食伤生财，一路相生
- 财格成立，无劫夺

**贫命特征**：
- 财多身弱，无力守财
- 比劫重而财轻，争财反败
- 财入墓库，财来财散

## 六、健康与命局

| 五行偏枯 | 对应健康问题 |
|---|---|
| 水少 | 肾虚、泌尿、血液不足 |
| 火少 | 心脏弱、循环不好、情绪低落 |
| 木少 | 肝胆、神经系统 |
| 金少 | 肺、大肠、皮肤 |
| 土少 | 脾胃、消化系统 |
| 水过旺 | 肾气散，水肿，寒湿 |
| 火过旺 | 心火亢，血压高，眼疾 |
| 土过旺 | 脾胃湿阻，肌肉病 |

## 七、韦氏命例分析方法

**标准论命步骤**：
1. 排出四柱（年月日时），标注大运
2. 判断日主强弱（重点看月令）
3. 取格局（月令透出）
4. 定用神（格局所需）
5. 看行运（与用神喜忌的关系）
6. 断具体事象（六亲、财官、健康）
7. 推应期（流年与大运交汇时）
